# HTTP
[HTTP](https://youtu.be/t1UtCblLk_0)

- hyperText Transfer Protocol의 줄임말
- 창시자 : 팀버너스리

## http의 역할
- 웹브라우저 -> 웹서버 (요청 Request)
- 웹서버 -> 웹브라우저 (응답 Response)

## 검사
- user-agent : 웹브라우저에 대한 정보
